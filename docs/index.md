# KTR-1 WEB教材

KTR-1はProMicro・電池ボックス・各種入出力を搭載したarduino制御学習用ロボットです。  
オープンソースハードウェアのため、書き込んであるプログラムや回路、基板の寸法etc......すべてを公開しています！  
現在arduinoの使い方がわからない方もKTR-1で遊んでもらって、arduino使いになっていただければ嬉しいです。
